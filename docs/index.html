Ligma OS

Give your hardware what it deserves.

But what is Ligma OS?

Ligma OS is a Linux® distribution that takes an asinine approach to everything it does.
It is a masochistic operating system for those who are tired of the ease of use of conventional operating systems.

What are the features of Ligma OS?

Ligma OS features an advanced set of features including:

  • No package manager
  • Minimal use of GNU utilities
  • A custom fork of the Linux® kernel
  • Musl libc
  • Busybox utilities
  • AI integration
